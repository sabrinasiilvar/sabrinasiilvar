# Sobre mim

Sou graduada em Análise e Desenvolvimento de Sistemas e atualmente pós-graduanda em Automação de Testes.

Iniciei minha transição de carreira em 2024, quando migrei da área de Segurança do Trabalho para a Tecnologia por meio do Premiersoft Academy. Desde então, venho construindo minha trajetória em Qualidade de Software, aliando minha formação acadêmica a uma rotina constante de aprendizado e prática em projetos.

Hoje, atuo como QA, com experiência em testes manuais e automatizados, validação de APIs e testes em aplicações web. Tenho como objetivo contribuir para a qualidade das entregas, identificando falhas, propondo melhorias e colaborando de forma próxima com os times de desenvolvimento.

## Competências

**Testes:** Manuais, Funcionais, API e Web  
**Automação:** Cypress, Playwright e Appium  
**API:** Postman  
**Linguagens Web:** HTML, CSS e JavaScript  
**Ferramentas de Gestão:** Jira, Azure DevOps e Trello  

## Em constante evolução ✨📈

Sigo em constante desenvolvimento, buscando aprimorar minhas habilidades técnicas e contribuir com soluções que gerem valor e qualidade para os projetos.
